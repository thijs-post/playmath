Plot a function

Explanation

Specify a function of x to be plotted, from x-min to x-max.

Supported formats are:

  • ASCIIMath -- e.g. 5x^4-2 , Never bother about multiplication symbols (like *)
  • LaTeX -- Just check the box!

Evaluation of the specified function is through Evaluatex.
Rendering of the formula is performed by KaTex.
